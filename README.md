WebStorm-Chinese
================


WebStorm Chinese Language Pack（中文语言包）

准备尝试使用WebStorm做开发，因为英文本身也不好，所以做了这个汉化包；

把压缩包里的"resources_zh_CN.jar"拷贝到WebStorm安装目录下的lib目录，重启即可！


因为开始使用了大量的自动翻译，虽然几经校对，加上英文水平有限，应该还有大量的翻译错误在里面！
